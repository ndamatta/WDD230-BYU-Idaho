White Space and Clean Desgin

Google

google.com

LIn this screenshot of the home page of the website Google, we can see the principle of white space and clean design applied. The main porpouse of the website is to have a search bar and a button to start the search, in this case that's the only thing we can see with the logo of the company. The design is minimalistic and it's not filling the entire website with large fonts or images that aren't not important for the main objective

PARC: Alignment

FamilySearch

familysearch.org

In this screenshot of the home page of the website FamilySearch, we can see the princple of alignment applied. We can appreciate a Centered Vertical Alignment in the nav, making all the elements fit in the same row. Also, we can appreciate a Centered Horizontal Alignment in the banner, taking the 'Get Started' button as reference

PARC: Contrast

Infobae

infobae.com

In this screenshot of the home page of the website Infobae, we can see the principle of contrast applied. The background of the website is white, the text is black and all other elements in the website have a good contrast ratio with the white background. Both, the text and the rest of the items are perfectly readable
